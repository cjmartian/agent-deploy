## Hello, World

My name is Connor Martin, and I'm a Senior Software Engineer at GitHub based in Seattle, WA. If you've used `conda install` to set up a Python environment, there's a chance you've run some code I helped maintain.

## My Journey

I started my career at **Anaconda**, where I worked across the entire conda Python distribution ecosystem. From maintaining the core `conda` package manager to managing dozens of conda-forge feedstocks (boto3, botocore, arrow-cpp, beautifulsoup4, and more), I got a deep appreciation for how critical package management infrastructure is to every developer's workflow.

At Anaconda, I also built tooling like the **anaconda-linter** for recipe quality assurance and managed CI/CD pipelines with Concourse CI. Working on a project used by millions of data scientists and developers taught me the importance of backward compatibility, thorough testing, and community engagement.

## What I'm Working On Now

At **GitHub**, I'm building developer tools and infrastructure. I'm particularly excited about the emerging **MCP (Model Context Protocol)** ecosystem — building the bridges that let AI agents interact with real-world infrastructure. I've been working on MCP server starters in both Go and Python.

## Why Open Source

Open source isn't just a development methodology for me — it's how I learn, how I connect with other engineers, and how I give back. Some of my favorite contributions have been to projects like:

- **thefuck** (95.8k stars) — I added conda command correction and git-lfs support
- **CPython** — Contributing to the language itself is something I'm proud of
- **Mattermost** — Building Go plugins for an open-source communication platform

## What's Next

I'm excited about the intersection of AI and developer tools. If you're working on something interesting in that space, I'd love to hear about it.
